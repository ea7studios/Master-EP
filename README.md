# Master-EP
This is going to be my best creation yet.
In the Master-EP, you can choose from 6 main categories: Pokedex, Regions, Anime, Gameplay, Routes and Locations, and Characters.
  Pokedex is going to possibly be the biggest sector with it splitting into the 9 regional pokedexes today: Kanto (Gen 1), Johto (Gen 2), Hoenn (Gen 3), Sinnoh (Gen 4), Unova (Gen 5), Kalos (Gen 6), Alola (Gen 7), Galar (Gen 8), and Paldea (Gen 9), with the games from each generation on the titel to show you the way. You can click on the pokedex to transport you the individual games, like for instance, if you click on Kanto, it will show Red/Blue and Yellow Pokedexes. Then you can click on either to see all the sprites for that Pokedex, which is really cool.
